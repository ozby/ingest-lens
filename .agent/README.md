# Agent directory

The authoritative operating contract for all contributors is [`AGENTS.md`](../AGENTS.md).
Read that first.

This directory contains the supporting detail that backs up that contract.

## Structure

| Path      | Purpose                                                       |
| --------- | ------------------------------------------------------------- |
| `guides/` | Operational policy: how agents should behave in this repo     |
| `rules/`  | Enforced rules per topic — each file is citable and diffable  |
| `skills/` | Repo-local skills: `$plan`, `$plan-refine`, `$pll`, `$verify` |

See [`guides/README.md`](./guides/README.md) and [`rules/README.md`](./rules/README.md) for indexes.
